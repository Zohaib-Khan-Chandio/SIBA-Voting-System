# SIBA Voting System

A Java console-based electronic voting system for university elections.

## Features
- CMS-ID validation
- One voter, one vote
- Department-specific candidates
- Winner declaration
- Admin controls
- Custom HashMap implementation

## Technologies
- Java
- Queue
- HashMap
- LinkedList
- PriorityQueue
